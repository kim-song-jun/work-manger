# 관리자 가이드

관리자는 매니저 권한 + **회사 전체 운영** 권한.

## 사이드바 (웹/데스크탑)

| 메뉴 | 용도 |
|---|---|
| 대시보드 | 오늘의 운영 KPI (출근률, 결원, 승인 대기) |
| 승인 | 회사 전체 승인 대기 — 일괄 승인/반려 가능 |
| 직원 | 직원 목록 검색, 권한/직책 변경, 비활성화 |
| 소멸 예정 | 30일 내 만료 연차 보유 직원 — 사용 권유 |
| 리포트 | 월별 출근/지각/초과근무 — CSV / PDF 내보내기 |
| 감사 | 시스템 변경 이력 (로그인, 권한 변경, 데이터 수정) |
| 코드 | 회사 코드 / 초대 링크 관리 |
| 52시간 | 주 52시간 컴플라이언스 (회사 전체 누적) |

## 직원 관리

### 신규 직원 등록 (CSV 일괄)
1. **직원** → **CSV 일괄 등록**
2. 양식 (`email,name,role,hired_at,department_name,position`) 다운로드 → 입력 → 업로드
3. **건조 실행 (dry-run)** 으로 검증 → 문제 없으면 **실행**.
4. 신규 가입자에게 비밀번호 재설정 메일 자동 발송.

### 권한 변경
- **직원** → 행 클릭 → **수정** → 역할 (EMPLOYEE / MANAGER / ADMIN / OWNER) 변경.

### 비활성화
- 퇴사자: **수정** → **비활성화**. 데이터는 보존, 로그인만 차단.

## 회사 코드

- **코드** → 신규 코드 발급 / 만료. 코드는 6자리 ASCII (회사 prefix 포함).
- 기본 1개 활성, 임시 채용 시즌엔 단기 코드 추가 발급.

## 리포트

### 월별 보고서
- **리포트** → 월 (YYYY-MM) 선택 → CSV / PDF 내보내기.
- 컬럼: 이름, 부서, 출근일수, 지각일수, 총 근무 분.

### 감사 로그
- **감사** → 액션 / 수행자 / 기간 필터.
- 90일 보존 (`apps/audit/` SOP-data-export-request 참조).

## 일괄 승인

- **승인** → 대기 항목 체크박스 선택 → **일괄 승인** / **일괄 반려**.
- 대량 처리 시 단일 round-trip 으로 처리 (50건도 1초 이내).

## 52시간 컴플라이언스

- **52시간** → 회사 전체 주간 누적.
- 주 52시간 초과 직원 명단 + 추세 그래프.
- 관리자가 미리 경고 발송 가능 (개별 알림).

## 데이터 export / 삭제 요청

- 직원의 GDPR / 개인정보보호법 요청은 **운영팀** SOP 따름:
  - [SOP — 데이터 export 요청](../operations/sop/sop-data-export-request.md)
  - [SOP — 데이터 삭제 요청](../operations/sop/sop-data-deletion-request.md)

## 보안

- 2FA TOTP 활성 (관리자 계정 필수 권장).
- 로그인 lockout: 5회 실패 → 30분 차단.
- 의심스러운 활동: 감사 로그 + Sentry 알림 모니터.
