# 매니저 가이드

매니저는 직원 권한 + **팀원 승인 권한** 을 가집니다.

## 매니저 인박스

- 상단 종 아이콘 → 인박스 → **승인 대기** 탭.
- 신청 종류: 연차 · 초과근무 · 출장 · 수동 출근.
- 빠른 결정: **승인** / **반려** 버튼. 반려 시 사유 입력 시트.
- 스와이프 제스처 (모바일): 좌→우 승인, 우→좌 반려.

## 팀 캘린더

- **팀** 탭 → 캘린더. 팀원 연차 / 출장 일정 한눈에.
- 색상 의미:
  - 🟢 출근
  - 🟡 외근
  - 🔵 연차
  - 🔴 결근

## 팀 보고서

- **팀** 탭 → **상세** → 월별 출근 / 지각 / 초과근무 합계.
- 본인 팀 데이터만 보입니다. 회사 전체는 관리자만.

## 권장 워크플로우

1. **출근 시간** — 인박스 점검 (밤사이 들어온 신청)
2. **점심 시간** — 팀 캘린더 확인 (오늘 결원/연차 파악)
3. **퇴근 전** — 다음 주 캘린더 확인 (인력 배치 미리)

## 알림 설정

- 매니저 전용 알림: 신청 도착 즉시 푸시.
- **마이** → **알림 설정** → "팀원 신청" 토글 활성 권장.
