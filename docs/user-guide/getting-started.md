# 처음 시작하는 분

## 1. 회사 코드 받기

회사 관리자가 발급한 **회사 코드** (예: `ACME` + 6자리 — `ACMEDM` 등) 가 필요합니다. 받지 못했다면 회사 인사 담당자에게 문의하세요.

## 2. 가입하기

1. 앱 또는 웹(`https://wm.example.com`) 에서 **시작하기** → **회사 코드 입력**
2. 회사 코드 입력 → 회사명 확인
3. 이메일 + 비밀번호 입력 → 인증 코드 메일 → 6자리 입력
4. 프로필 (이름, 부서, 직급) 입력
5. 출근 위치 선택 (본사, 지점, 재택)
6. 알림 권한 허용 (선택)
7. 위젯 추가 (선택, iOS / Android / Windows)

가입 완료 후 자동으로 메인 화면으로 이동합니다.

## 3. 첫 출근

1. 메인 화면 가운데 큰 **밀어서 출근** 버튼
2. 위치 권한 요청 → 허용
3. 출근 위치 자동 감지 (지오펜스 안에 있으면 자동)
4. 출근 처리 완료 → 화면 상단 KPI 가 업데이트됨

## 4. 첫 알림

오늘 일정이 있는 첫 알림 (예: 회사 공지, 매니저 메시지) 이 푸시로 도착합니다. 알림이 도착하지 않으면 [FAQ — 알림 설정](faq.md#알림-설정) 참조.

## 5. 다음에 할 일

- 직원이라면 → [employee.md](employee.md) — 연차/초과근무 신청 방법
- 매니저라면 → [manager.md](manager.md) — 팀원 승인 방법
- 관리자라면 → [admin.md](admin.md) — 직원 관리 / 리포트
