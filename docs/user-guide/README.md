# 사용자 가이드

근무 관리 시스템 사용 매뉴얼. 역할별로 필요한 부분만 읽으세요.

## 시작하기

- [처음 시작하는 분](getting-started.md) — 회사 코드 입력 → 가입 → 첫 출근
- [자주 묻는 질문](faq.md) — 비밀번호 재설정, 위치 권한, 알림 설정 등

## 역할별 가이드

| 역할 | 가이드 | 주요 기능 |
|---|---|---|
| 직원 (EMPLOYEE) | [employee.md](employee.md) | 출근/퇴근, 연차 신청, 초과근무 신청, 알림 |
| 매니저 (MANAGER) | [manager.md](manager.md) | 팀원 승인, 팀 캘린더, 매니저 인박스 |
| 관리자 (ADMIN) | [admin.md](admin.md) | 회사 직원 관리, 회사 코드, 리포트, 감사 로그, 52시간 컴플라이언스 |
| 소유주 (OWNER) | [owner.md](owner.md) | 회사 설정, 알림 정책, 데이터 내보내기/삭제 |

## 문제 해결

- 앱이 위치를 인식하지 못해요 → [FAQ — 위치 권한](faq.md#위치-권한)
- 푸시 알림이 안 와요 → [FAQ — 알림 설정](faq.md#알림-설정)
- 비밀번호를 잊었어요 → [FAQ — 비밀번호 재설정](faq.md#비밀번호-재설정)
- 다른 회사로 이직했어요 → [FAQ — 회사 코드 변경](faq.md#회사-코드-변경)
