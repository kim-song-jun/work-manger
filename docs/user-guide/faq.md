# 자주 묻는 질문 (FAQ)

## 일반

### 비밀번호 재설정
- 로그인 화면 → **비밀번호를 잊으셨나요?** → 회사 이메일 입력 → 메일 링크 클릭 → 새 비밀번호.
- 메일이 안 오면: 스팸 폴더 확인 → 그래도 없으면 회사 IT 에 문의 (관리자가 [SOP — 긴급 비밀번호 재설정](../operations/sop/sop-emergency-password-reset.md) 으로 처리 가능).

### 회사 코드 변경
- 이직 시: 기존 회사에서 비활성 처리 후 새 회사 코드로 신규 가입 (동일 이메일 사용 가능).
- 회사 코드를 잘못 알아서 다른 회사로 가입한 경우 → 운영팀에 문의 (수동 정정 필요).

### 2FA (TOTP) 분실
- 관리자 / 소유주 계정만 강제. 인증 앱 분실 시 백업 코드로 일회 로그인.
- 백업 코드 분실 시: 신원 확인 후 운영팀이 2FA 초기화.

## 위치 / 출근

### 위치 권한
- iOS: **설정** → **근무 관리** → **위치** → **앱 사용 중에 허용** (또는 **항상 허용**).
- Android: **설정** → **앱** → **근무 관리** → **권한** → **위치** → **항상 허용** (지오펜스 백그라운드).
- 데스크탑: 트레이 자동 출근은 OS 위치 서비스 + 회사 Wi-Fi BSSID 매칭.

### 출근 위치가 안 잡혀요
- GPS 신호 약함: 창가로 이동 → 1분 대기.
- 지오펜스 반경 밖: 회사 관리자가 반경을 충분히 설정했는지 확인.
- 그래도 안 되면: **수동 출근** 으로 사유 + 매니저 승인.

### 위젯 1탭 출근이 안 돼요
- iOS: 위젯 추가 후 앱을 한 번 실행해야 활성.
- Android: 배터리 최적화 OFF 권장 (`설정` → `배터리` → `근무 관리` → `최적화 안 함`).

## 알림 설정

### 푸시 알림이 안 와요
1. **마이** → **알림 설정** → **알림 받기** 토글 ON
2. OS 알림 권한 (위치 권한과 별도) 허용
3. iOS: 백그라운드 새로고침 활성
4. Android: 배터리 최적화에서 앱 제외
5. 그래도 안 되면 회사 IT 에 문의 (Web Push VAPID / APNs 키 만료 가능성)

### 너무 많은 알림
- **마이** → **알림 설정** → 카테고리별 토글 (승인 / 연차 / 공지 / 시스템).
- 매니저: "팀원 신청" 만 활성, 시스템 알림 OFF 가능.

## 매니저

### 인박스에 신청이 안 보여요
- 본인이 매니저로 지정된 직원의 신청만 보임. 회사 조직 구조 (`매니저-직원` 매핑) 확인.
- 관리자에게 매핑 수정 요청.

### 일괄 승인
- 모든 신청 종류 (연차/초과근무/출장/수동 출근) 가 한 번에 승인 가능 (관리자 권한 시).
- 매니저는 본인 팀원만, 관리자는 회사 전체.

## 데이터

### 내 출근 기록 export
- **마이** → **데이터 내보내기** → CSV / JSON. 본인 데이터만.
- 회사 전체 export 는 관리자/소유주 권한.

### 데이터 삭제 (계정 탈퇴)
- 직원: 회사 IT 에 퇴사 처리 요청 → 관리자가 비활성화 → 14일 유예 후 PII 익명화.
- [SOP — 데이터 삭제 요청](../operations/sop/sop-data-deletion-request.md).

## 기술

### Electron 앱 자동 업데이트
- 자동 (S3 publish + electron-updater). 새 버전 다운로드 후 다음 재시작 시 적용.
- 수동 확인: **마이** → **버전** → **업데이트 확인**.

### 오프라인 모드
- 모바일: 출근 / 퇴근 1회 기록은 오프라인 큐 → 온라인 복귀 시 동기화.
- 웹 / 데스크탑: 오프라인 미지원.

## 문의

- 사용 중 문제: 회사 IT 헬프데스크.
- 버그 리포트: 운영팀 → GitHub Issues (회사 SOP 따름).
- 보안 취약점: 비공개 채널 (`security@<회사>`) — 즉시 alert.
