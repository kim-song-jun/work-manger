# 직원 가이드

## 출근 / 퇴근

### 일반 출근
- 메인 → **밀어서 출근**. 위치 권한 허용 시 자동 위치 인식.
- 본사 / 지점 / 재택 중 선택 (관리자가 활성한 옵션만 표시).

### 출근을 깜빡했을 때 (수동 출근)
- 메인 → **출근 추가** → 날짜 선택 + 사유 입력 → 매니저 승인 후 반영.

### 외근 / 거래처 출장
- 메인 하단 탭 **출장** → 일정 입력 → 매니저 승인 → 자동 출근 처리.

## 연차

### 잔액 확인
- 하단 **연차** 탭 → 잔여 / 사용 / 발생 / 소멸 예정 카드.
- "소멸 예정" 이 있으면 만료 전에 사용하세요.

### 연차 신청
1. **연차** 탭 → **연차 신청**
2. 캘린더에서 시작일/종료일 클릭 (반차는 시작=종료, 반차 옵션 선택)
3. 사유 입력 → **신청하기**
4. 매니저 승인 후 자동 차감.

### 신청 취소
- **연차** 탭 → 내 신청 목록 → **취소** (승인 전만 가능).

## 초과근무

- **초과근무** 메뉴 → 날짜 + 분 (1분 ~ 720분) + 사유 입력 → 매니저 승인.
- 회사가 주 52시간 컴플라이언스 활성 시 한도 초과 신청 차단.

## 알림

- 상단 종 아이콘 → 인박스. 필터: 전체 / 승인 / 연차 / 공지.
- 모든 항목 읽음 처리: **모두 읽음**.

## 팀 / 동료 상태

- **팀** 탭 → 팀원 출근 상태 (출근/퇴근/연차/외근).
- 팀 캘린더에서 다음 주 연차 일정 확인.

## 회사 공지

- **공지** 탭 → 핀 고정된 공지 + 카테고리 (정책/이벤트/일반).

## 위젯 (Android / iOS / Windows / macOS)

- 홈 화면에 **출근 위젯** 추가 → 앱 진입 없이 1탭 출근.
- iOS: WidgetKit small / medium / large.
- Android: Glance widgets.
